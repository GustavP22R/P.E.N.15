This program requires the following
 - node.js
 - JavaScript Compiler
 - Ability to local host

For the program to run do you need to do following
1. Open a terminal
2. Change directory to the socket server
3. Run the server using node.js with the following command: node server.js
4. You should get a link where to local host is located
5. Join with 2 clients to communicate
6. Server can be shut down with 'ctrl + c' in the terminal
